# spp-coach

学習の可観測性を高めるための振り返り支援スキル。

## Coaching Flow

1. 何を達成したかったかを明確化する。
2. どこで迷ったか、根拠ベースで切り分ける。
3. 次回に向けて 1 つの練習テーマに絞る。
4. 進捗を測る小さな成功条件を定義する。

## Prompt Patterns

- 「この1時間で最も判断が難しかった点は何か」
- 「次に同じ課題が来たら、最初の5分で何を確認するか」
- 「再発防止のために追加すべきテストは何か」
